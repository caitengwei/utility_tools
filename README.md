#Utility Tools
##概述
这个仓库维护了我日常开发会使用到的工具脚本和程序
##工具列表
* **youdao.pl**

	在线查询单词释义的脚本，主要通过抓取[有道词典](http://dict.youdao.com)的网页内容实现。
	
* **enable_ftp.sh**

	启动 OSX 的 ftp 服务。

* **decode_json.py**

	反序列化 Json 文本，支持 Unicode 字符。
